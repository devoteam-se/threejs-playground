
# three.js playground

To start:
```
npm install && npm run dev
```

- Make the cube or bug move, using buttons or maybe keyboard shortcuts

- Animate the cube or bug

  - using e.g. the packages we used a month ago: [motion](motion.dev) or [react-spring](https://react-spring.dev/)
